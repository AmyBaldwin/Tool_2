Downloading the output files

To transfer the files to your personal computer drag and drop.

Each time Tool 2 runs it makes a new folder where all of the output files are stored. The new output folder can be found inside the Tool 1 folder, inside the folder named output_tool_1:

/home/ubuntu/genome_store/Tool_2_custom/output_tool_2/xxx-yy-zz_YYYY-MM-DD_HH:MM/

The name of the new folder includes the ABRicate database you specified when you started the run (xxx), as well as the identity and coverage settings (yy and xx). This is followed by the date and time.
